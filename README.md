# Generative AI Research Project

This repository organises materials for the REIT6811 research scenario: **Using Generative AI Tools – Boon or Bane**.

## Repository Structure

- `01_Literature_Review/`  
  Journal articles, conference papers, books, newspaper articles, and literature review notes.

- `02_Quantitative_Analysis/`  
  Survey questions, survey data, analysis scripts, CSV/Excel files, and quantitative analysis reports.

- `03_Qualitative_Analysis/`  
  Interview transcripts, interview protocols, consent forms, qualitative analysis reports, insights, and visualisations.

- `04_Drafts_and_Reports/`  
  Draft research proposals, conference papers, final reports, and written outputs.

- `05_Additional_Materials/`  
  Information sheets, photos, media files, and other supporting materials.

## File Naming Convention

Files should use the following naming structure:

`YYYYMMDD_ProjectName_DataType_Description_Version`

Example:

`20260424_GenAIResearch_SurveyData_Cleaned_v1.csv`

## Collaboration Guidelines

1. Use clear and descriptive file names.
2. Write meaningful commit messages.
3. Do not upload sensitive or identifiable participant data to public GitHub repositories.
4. Store consent forms, interview transcripts, survey responses, photos, and identifiable data in secure restricted-access storage.
5. Update the README if the project structure changes.
